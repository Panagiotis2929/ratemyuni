# RateMyUni ★
**Η απόλυτη πλατφόρμα αξιολόγησης ακαδημαϊκών για φοιτητές στην Ελλάδα.**

### Σκοπός του Έργου
Το **RateMyUni** είναι μια PWA (Progressive Web App) που γεφυρώνει το χάσμα επικοινωνίας μεταξύ φοιτητών και ακαδημαϊκού περιβάλλοντος. Στόχος της πλατφόρμας είναι η παροχή αντικειμενικής πληροφόρησης, η ανταλλαγή εμπειριών και η βελτίωση της φοιτητικής καθημερινότητας μέσα από δεδομένα.

---

### Κύρια Χαρακτηριστικά
* **Ασφαλές Ακαδημαϊκό Οικοσύστημα:** Υποχρεωτική επαλήθευση μέσω `.edu.gr` email (Supabase integration) για την εξασφάλιση αυθεντικών κριτικών.
* **Σύστημα Σύγκρισης 1v1:** Δυνατότητα άμεσης, απευθείας σύγκρισης καθηγητών με βάση πολλαπλά κριτήρια αξιολόγησης.
* **Data-Driven Analytics:** * Line charts εξέλιξης βαθμολογίας ανά εξάμηνο.
    * Dashboard στατιστικών και Rankings ανά ΑΕΙ/Τμήμα.
    * Trending καθηγητές με βάση τη δραστηριότητα των τελευταίων 7 ημερών.
* **Smart Features:** * Αυτόματα "Badges" (π.χ. *Οργανωμένος*, *Εμπνέει*, *Top-Rated*).
    * Προηγμένη αναζήτηση και φιλτράρισμα.
    * Δυνατότητα Export αναφορών σε PDF.
* **Mobile-First Design:** PWA υποστήριξη για native-like εμπειρία στο κινητό και Dark Mode για βέλτιστη αναγνωσιμότητα.

---

### Τεχνολογίες (Tech Stack)
* **Frontend:** HTML5, CSS3 (Custom Design System), JavaScript (Vanilla).
* **Backend & DB:** Supabase (PostgreSQL, Authentication, Row Level Security).
* **PWA:** Service Workers, manifest.json.
* **Data Visualization:** Chart.js / Custom Implementation.

---

### Γιατί RateMyUni;
Σε ένα ακαδημαϊκό περιβάλλον που συνεχώς εξελίσσεται, οι φοιτητές χρειάζονται διαφάνεια. Το RateMyUni δεν είναι απλώς μια λίστα αξιολογήσεων, είναι ένα εργαλείο που βοηθά τους φοιτητές να πάρουν τεκμηριωμένες αποφάσεις για την πορεία των σπουδών τους.
